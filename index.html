The Parklands of Floyd's Fork

By:Jack-Harold Enyeart

The goal of this project is to display the beautiful park of the 'The Parklands of Floyd's Fork' in ways that cannot be seen in traditional maps. The way this is being accomplish is by displaying sections of the park in the form of point cloud data, virtual video, and through a contour map.

The Parklands of Floyd's Fork is a nearly 4000 acre public park system under developement along the Floyd's Fork watershed in Louisville. Louisville is using the parkland to proactively shape future development and are working to develop a more sustainable model for urban growth beyond the city limits, in which the Parklands is the green frontier of the regionâ€™s eastward development and the future green infrastructure of communities yet to be. (Source:https://www.wrtdesign.com/work/the-parklands-of-floyds-fork)
Using point cloud data, a realistic 3D presentation can be made using cesium.com
Using elevation data, a contour map is used tell the shape of the land surface of the Humana Grand Allee
Key: contour every 10 feet, blue lines indicate 100 feet
The video displays the entrance to the Coppiced Woods Trail, one of the many different trails available in the park

Using python and ArcGIS pro, lidar data was used to make a point cloud data set around the specified area of 'The Parklands of Floyd's Fork' for use in online resource Cesium, in addition this point data and ArcGIS pro were used to make the animation. For the contour map, a combination of 2018 NAIP aerial color photography and lidar data points were used to estimate the elevtions of the specified area, also 'The Parklands of Floyd's Fork', in order to create this map type.

The data and methods used to create the assets were Lidar point cloud data from the KyFromAbove, 2018 NAIP aerial color photography from USGS EarthExplorer, Google Maps longitude and latitude points for reference points
